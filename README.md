# Nasa Collection

Single Page Aplicattion usando React + Typescript

## Descrição

Simples Spa que utiliza a api da nasa para pesquisar imagens e videos


### Dependencias utilizadas

* hooksy
* semantic-ui-react
* semantic-ui-css

### Instalação

* git clone https://github.com/malltapps/nasa-spa.git
* yarn ou npm i
